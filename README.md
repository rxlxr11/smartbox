# SmartBox 3.1

SmartBox 3.1 是一款面向日常效率与轻娱乐的跨平台小程序，围绕“记录、管理、反馈、激励”四个核心场景，整合了任务管理与游戏训练等能力，帮助用户在碎片化时间里持续完成自我管理。

## 项目定位

这个项目的目标不是单一工具，而是一个持续扩展的个人成长入口。它希望让用户在同一个应用里完成：

- 日常任务与习惯管理
- 数据化反馈与趋势观察
- 轻量化专注训练
- 后续可扩展的笔记、工具、AI 规划与旅行记录能力

## 当前功能状态

### 已实现

#### 1) 习惯打卡与任务管理

用户可以进行日常任务与习惯的创建、编辑、完成管理，并支持不同任务类型下的互动逻辑。整体流程覆盖了“添加任务 → 日常执行 → 状态更新 → 历史复盘”的完整闭环。

#### 2) 数据统计与可视化反馈

已提供任务与习惯相关的统计展示能力，帮助用户快速了解近期执行情况、连续性表现与阶段趋势。

#### 3) 登录与账户体系

已接入账户登录能力，支持用户在多端保持个人数据一致，保障个人记录的连续性和可追踪性。

### 开发中

#### 游戏中心

游戏中心已具备基础能力并上线首个训练型游戏（舒尔特方格）。当前版本支持成绩记录与个人历史表现查询，后续会继续扩展更多游戏内容与玩法。

### 规划中

- 随手笔记
- 实用工具集合
- AI 计划与智能排序
- 记忆旅游（路线与足迹记录）

这些模块将按“可用优先、体验优先、可持续迭代”的原则逐步落地。

## 产品特点

- **聚合体验**：把任务、习惯、训练等高频场景收敛在一个应用内。
- **反馈驱动**：不仅记录行为，还强调趋势反馈与结果感知。
- **轻量上手**：核心流程短、操作路径清晰，适合日常快速使用。
- **持续演进**：以模块化方式迭代，便于后续功能扩展。

## 适用人群

- 希望养成稳定习惯的人
- 需要管理每日任务与进度的人
- 希望通过轻量训练提升专注力的人
- 喜欢在一个应用中统一管理个人成长内容的人

## 平台支持

当前主要面向微信小程序与 H5 场景，后续会结合实际使用反馈持续优化多端体验。

## 版本说明

SmartBox 3.1 目前处于持续迭代阶段。我们会优先保证已上线功能的稳定性，同时逐步推进新模块上线，确保每次更新都能带来可感知的体验提升。
